# 💼 Portfolio - Nadiah Mauricio

> Desenvolvedora Full Stack & Analista de Dados em transição para a área tech

## 🌟 Sobre o Projeto

Portfolio pessoal desenvolvido em React com design moderno e responsivo, apresentando minhas habilidades, projetos e experiência em desenvolvimento web e análise de dados.

## 🔗 Demo

**🌐 [Ver Portfolio Online](https://nmauricio.netlify.app/)**

## ✨ Funcionalidades

- 🎨 Design moderno e responsivo
- 🌍 Suporte a idiomas (PT/EN)
- ⚡ Animações suaves com Framer Motion
- 📱 Totalmente responsivo
- 🎯 Seções organizadas (Sobre, Skills, Projetos, Formação, Contato)
- 🔗 Links funcionais para redes sociais
- 📄 Download de CV

## 🛠️ Tecnologias Utilizadas

- **Frontend:** React.js
- **Styling:** CSS3 com animações
- **Animações:** Framer Motion
- **Ícones:** Lucide React
- **Deploy:** Netlify
- **Versionamento:** Git & GitHub

## 🚀 Como Executar Localmente

```bash
# Clone o repositório
git clone https://github.com/Nad1ah/portfolio_done.git

# Entre no diretório do projeto
cd portfolio_done/nadiah-portfolio

# Instale as dependências
npm install

# Execute o projeto
npm start

# Acesse http://localhost:3000
